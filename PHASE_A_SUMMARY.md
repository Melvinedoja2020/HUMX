### ✅ Design System Mapped

**Primary Colors:**

- Page Background: `#ecece9` (warm cream/beige)
- Card Background: `#ffffff` (pure white)
- Primary Text: `#111111` (near black)
- CTA/Black Accent: `#1a1a1a` (deep black)
- Borders: `#dddddd` (light gray)

**No bright accent colors. NO gold, NO blue. Minimal palette.**

**Typography:**

- Font: Manrope (400, 500, 600, 700, 800)
- Hero H1: 48-60px (text-4xl to text-5xl, semibold)
- H2/Section: 36-48px (text-3xl to text-4xl, semibold)
- Body: 16px (text-base, regular)
- Labels: 12-14px (text-xs to text-sm, semibold)
- All text is tight line-height on headings, relaxed (1.75) on body

**Spacing:**

- Section vertical: 64px (py-16)
- Card padding: 20-24px (p-5 or p-6)
- Horizontal: 16px mobile / 24px tablet / 40px desktop (px-4 / px-6 / px-10)
- Grid gaps: 16px (gap-4)

**Shadows:**

- Cards: soft shadow (shadow-sm)
- Drawer: large shadow (shadow-2xl)
- Buttons: NO shadow (flat)

**Border Radius:**

- Buttons/Inputs: 12px (rounded-xl)
- Cards (medium): 16px (rounded-2xl)
- Cards (large): 24px (rounded-3xl)
- Pills: full (rounded-full)

**Grid System:**

- Mobile: 1 column
- Tablet (md 768px): 2 columns
- Desktop (lg 1024px): 3 columns
- Container max: 1280px (max-w-7xl)

**Animations:**

- Reveal: 600ms, cubic-bezier, opacity + Y-translate
- Hover: 300ms ease, slight scale or lift
- Drawer: 300ms spring, X-translate from right
- Stagger: index × 60ms delay
